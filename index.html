Enter Your FRED API Key

This tool uses the Federal Reserve's FRED API to fetch economic data. Each user needs a free API key. Here's how to get one:

1. Create a free account at fred.stlouisfed.org
2. Log in → My Account → API Keys
3. Click “Request API Key” — it’s generated instantly

The account and key are completely free and take about 2 minutes to set up.

How to Use This Tool

What is this?

This tool lets you search, visualize, and export economic data from FRED — the Federal Reserve Bank of St. Louis’s database of 800,000+ economic time series.

Getting started

  • Type a series name or ID in the search bar (e.g. “unemployment” or “UNRATE”)
  • Or click any of the pre-loaded chips below the search bar
  • Use the controls to adjust units, frequency, and time range

Controls explained

  • Units — how the data is expressed (level, % change, etc.)
  • Frequency — how often data points appear (monthly, quarterly, etc.)
  • Time Range — preset buttons (1Y, 3Y, 5Y…) or custom start/end dates
  • Y-Axis Range — manually set min/max to zoom in on a specific range
  • Line / Bar — switch between chart types

Exporting

  • Export PDF — downloads a formatted PDF with the Baird logo, chart, and stats
  • Add to Deck — adds the current chart as a slide to a PowerPoint deck
  • Export Deck (.pptx) — downloads all slides as a PowerPoint file
  • Type a custom title in the blue header bar before exporting
  • Add your team name in the field on the right of the header

Other features

  • Share Chart — generates a URL that opens this exact chart view
  • Hide Avg. Line — toggle the dashed average reference line
  • Move Avg. Label — cycles the label through positions to avoid overlap
  • Series ID badge — click it in the meta line to copy the series ID

About FRED

FRED is maintained by the Federal Reserve Bank of St. Louis and is free to use. Each user needs their own free API key — get one here.
